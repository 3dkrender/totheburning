# EOSIO Singleton Tables

This is an exercise to explain how to create and modify singleton tables in eosio smart contracts. 
Explanations: [Using eosio singleton tables (in WAX Blockchain)]()

Created by Marcos DK - 3DK Render WAX Guild

# Build

```cd <smart_contract_directory>
git clone
go to directory
md build
cd build
cmake ..
make
```

## Utility

- This smart contract will destroy all NFTs transferred to it.
- Accepts SimpleAssets and Atomic format NFTs (WAX Blockchain).
- Destroying NFTs is irreversible!
